# 党务工作智能问答助手

**部门**: 党群工作部  
**序号**: 4

## 原始需求描述

围绕党务场景建设智能问答助手，核心能力包括：

1. 结构化知识库构建：整合政策文件、操作流程等核心内容。
2. 7x24小时智能问答：支持自然语言交互，解答各类党务咨询。
3. 个性化知识推荐：根据用户岗位和历史记录推荐相关文件。
4. 自动知识更新：定期抓取最新政策，保持知识库时效性。

业务价值：减少重复性咨询处理时间，提高工作效率；扩展可定制自学渠道，支持党务工作者随时开展业务学习。

## 应用定位

党务智答中枢是一个面向党务管理场景的演示型管理控制台，采用 Mock 数据闭环，覆盖知识库构建、问答服务运营、个性化推荐和智能助手交互。

## 技术栈

- Next.js App Router + React Hooks + TypeScript strict
- Ant Design（管理台 UI）
- Ant Design X（`Sender` / `Bubble` / `Conversations`）
- Prisma + SQLite（本地演示数据持久化）

## 本地启动

1. 安装依赖：`npm install`
2. 初始化数据库：`npm run db:init`
3. 启动开发服务：`npm run dev`
4. 打开 `http://localhost:3000`

## 常用命令

- `npm run verify`：校验目录结构、核心组件和文档约束
- `npm run typecheck`：TypeScript 严格类型检查
- `npm run build`：生产构建检查
- `npm run db:seed`：重置并写入党务场景 Mock 数据

## 数据闭环说明

- 首屏通过 `lib/service.ts` 加载 SQLite 数据，空库自动写入专业演示数据。
- 新增条目、状态推进、删除确认、智能问答采纳均走 Server Actions。
- 高优先级任务推进和删除操作均带二次确认与状态反馈。
- 政策文库、流程手册、历史问答、政策订阅均使用 `lib/mock-integrations.ts` 模拟。
